# Chickpea-Android

Media runtime in C++/Javascript. MIT License.

# Supported platforms

 * Android 4+ with arm v7a CPU

# Libraries used

 * [JXCore](https://github.com/jxcore/jxcore) - MIT License

# Requirements for building

 * JDK 7
 * android-sdk
 * android-ndk
 * Windows

# Important notes

Environment variables have to be set before using scripts.
